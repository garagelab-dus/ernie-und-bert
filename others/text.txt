OpenArmLeaderConfigBase ['teleop.left_arm_config']:
  Base configuration for the OpenArms leader/teleoperator with Damiao motors.

  --teleop.left_arm_config.port str
                        CAN interfaces - one per arm Arm CAN interface (e.g.,
                        "can3") Linux: "can0", "can1", etc. (default: None)
  --teleop.left_arm_config.can_interface str
                        CAN interface type: "socketcan" (Linux), "slcan"
                        (serial), or "auto" (auto-detect) (default: socketcan)
  --teleop.left_arm_config.use_can_fd bool
                        CAN FD settings (OpenArms uses CAN FD by default)
                        (default: True)
  --teleop.left_arm_config.can_bitrate int
                        Nominal bitrate (1 Mbps) (default: 1000000)
  --teleop.left_arm_config.can_data_bitrate int
                        Data bitrate for CAN FD (5 Mbps) (default: 5000000)
  --teleop.left_arm_config.motor_config Dict
                        Motor configuration for OpenArms (7 DOF per arm) Maps
                        motor names to (send_can_id, recv_can_id, motor_type)
                        Based on:
                        https://docs.openarm.dev/software/setup/configure-test
                        OpenArms uses 4 types of motors: - DM8009
                        (DM-J8009P-2EC) for shoulders (high torque) - DM4340P
                        and DM4340 for shoulder rotation and elbow - DM4310
                        (DM-J4310-2EC V1.1) for wrist and gripper (default:
                        {'joint_1': (1, 17, 'dm8009'), 'joint_2': (2, 18,
                        'dm8009'), 'joint_3': (3, 19, 'dm4340'), 'joint_4':
                        (4, 20, 'dm4340'), 'joint_5': (5, 21, 'dm4310'),
                        'joint_6': (6, 22, 'dm4310'), 'joint_7': (7, 23,
                        'dm4310'), 'gripper': (8, 24, 'dm4310')})
  --teleop.left_arm_config.manual_control bool
                        Torque mode settings for manual control When enabled,
                        motors have torque disabled for manual movement
                        (default: True)
  --teleop.left_arm_config.position_kp List
                        TODO(Steven, Pepijn): Not used ... ? MIT control
                        parameters (used when manual_control=False for torque
                        control) List of 8 values: [joint_1, joint_2, joint_3,
                        joint_4, joint_5, joint_6, joint_7, gripper] (default:
                        [240.0, 240.0, 240.0, 240.0, 24.0, 31.0, 25.0, 16.0])
  --teleop.left_arm_config.position_kd List
